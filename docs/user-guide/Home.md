# Get OPIR

Get OPIR is the personal desktop edition of OPIR. It is a local-first workspace for planning project outcomes, tasks, and timeline work without accounts or servers.

Current version: `v0.3.2 beta`

## Core Idea

OPIR means:

- Output: the result that should exist.
- Process: the work path used to create the result.
- Input: the information, people, materials, or constraints needed.
- Revision: changes made when the plan no longer matches reality.

## Personal Edition

This edition is for individual desktop use.
