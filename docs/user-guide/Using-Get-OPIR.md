# Using Get OPIR

## Projects

A project stores the main outcome and a list of tasks. Use one project for one meaningful result.

## Tasks

Each task includes:

- title;
- owner;
- status;
- priority;
- start date;
- due date;
- output;
- process;
- input;
- notes.

## Timeline

The timeline sorts tasks by due date and gives a compact view of the current sequence of work.

## Backup

Use Export to download a JSON backup of your local workspace. Use Import to restore or move that workspace to another machine.
